# Assignment Portal - Frontend

react app for assignment portal thing

## how to run

```
npm install
npm run dev
```

opens on localhost:5173

## pages
- login page
- teacher dashboard (create/publish assignments)
- student dashboard (view and submit answers)

## test users

## tech
- react
- tailwind css
- vite
